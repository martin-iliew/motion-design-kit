# Sticky Proof Strip

**Framework:** HTML/CSS
**Category:** proof
**2026 Relevance:** Major landing pages still show proof early, but the best ones keep it compact and integrated instead of turning it into a giant brand wall.

## Description

Sticky Proof Strip is a compact row of customer logos, metrics, or trust bullets positioned immediately below the hero CTA cluster or anchored near the transition into the next section.

## Do's

- keep the strip compact
- use real proof, not filler badges
- align it tightly with the hero copy or first section boundary

## Don'ts

- do not create a logo cemetery
- do not repeat the same proof again two sections later
- do not let it become the most visually dominant element

## Best Practices

This pattern works because it reduces early skepticism without slowing the scan. It should reassure, not distract.
