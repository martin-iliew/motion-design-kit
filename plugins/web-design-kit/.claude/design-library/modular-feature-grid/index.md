# Modular Feature Grid

**Framework:** HTML/CSS
**Category:** features
**2026 Relevance:** Modular grids remain current because they package complexity cleanly, but the strongest pages use them with restraint instead of making every card a mini-poster.

## Description

Modular Feature Grid organizes features into a disciplined card system with light variation in span, emphasis, and supporting visuals.

## Do's

- group features by job to be done
- vary only one or two cards for hierarchy
- keep headings short and outcome-led

## Don'ts

- do not turn every card into a visual experiment
- do not overuse gradients, shadows, and decorative chrome
- do not mix unrelated hierarchy systems inside the same grid

## Best Practices

Think "controlled bento", not gallery bait. If the grid feels louder than the hero, pull it back.
