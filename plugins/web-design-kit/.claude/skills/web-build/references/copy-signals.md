# Copy Signals

Use these to keep the layout conversion-ready.

## Headlines

- say what the product is
- say who it is for when useful
- say the primary outcome

## Subheads

- explain the mechanism or differentiator
- avoid generic "all-in-one" filler unless it is immediately specified

## CTA labels

- `Start free`
- `Book demo`
- `See it in action`
- `Talk to sales`

Avoid weak labels like `Learn more` for the primary CTA.

## Proof labels

- customer count
- revenue impact
- time saved
- deployment speed
- security or compliance markers
