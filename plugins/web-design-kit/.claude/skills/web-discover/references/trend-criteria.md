# Trend Criteria

Treat a pattern as worthy of the library only when most of these are true:

- it appears on current official landing pages from established product companies
- it improves clarity, proof, or conversion flow
- it scales across responsive breakpoints
- it can be implemented in ordinary frontend stacks
- it does not depend on novelty-first styling to feel current

Reject patterns that are mostly social-post bait, gallery bait, or editorial experiments.
