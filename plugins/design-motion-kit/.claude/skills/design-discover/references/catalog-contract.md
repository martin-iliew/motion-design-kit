# Catalog Contract

Each pattern folder must contain:

- `index.md`
- `spec.yaml`
- `composition.yaml`

Each catalog entry must include:

- `id`
- `name`
- `category`
- `framework`
- `surfaces`
- `site_contexts`
- `folder`
- `composition`
- `description`
- `trend_score`
- `status`
- `conversion_signal`
- `last_reviewed`
- `sources`
- `manual_override`
- `notes`
